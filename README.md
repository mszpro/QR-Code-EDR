# QR-Code-EDR

Using the new Metal features and EDR support, only highlight (brighten up) the QR code on the screen

WWDC 2022で発表されたMetalの新機能では、EDRを使って画面画像の一部分だけを明るく（現在設定されている画面の明るさ以上）することができます。
